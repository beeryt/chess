# Chess
A Chess program inspired by [Coding Adventure: Chess AI](https://youtu.be/U4ogK0MIzqk).

## Installation
You can use cmake to build chess.
```bash
mkdir build && cd build
cmake ..
make -j8
```

## Usage
```bash
chess [FEN]
```
